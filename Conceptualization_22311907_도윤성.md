## 1. Business purpose
### 1.1. Project background

<img width="500" height="300" alt="image" src="https://github.com/user-attachments/assets/6694c2b6-67ff-43c0-a13f-8ace4049babd" />
<img width="189" height="320" alt="image" src="https://github.com/user-attachments/assets/b1b4b820-39a9-46ce-b6de-623b369550ab" />

2024년 12월 기준, 국내 상장법인 주식을 보유한 개인투자자는 1,410만 명에 도달하며 '전국민 투자 시대'가 도래했습니다. 이는 전체 투자자의 99.1%를 차지하는 수치로 개인투자자의 수가 증가했음을 알려줍니다. 그러나 이러한 양적 성장에도 불구하고 정보의 질적 활용 측면에서는 다음과 같은 한계가 존재합니다.

+ 정보의 과부하와 결정 마비: 2025년 기준 조사에 따르면 인터넷 이용률은 94.9%에 달하며, 그 중 모바일 인터넷 뉴스 이용률은 70.9%로 전통 매체를 완전히 대체하고 있습니다. 또한, 투자자의 61.9%가 뉴스가 투자에 도움이 된다고 응답했으나 하루에 쏟아지는 수만 건의 기사 중 핵심 정보를 선별하는 데 소요되는 시간적 비용이 기회비용을 초과하는 정보 과부화 상태에 직면해 있습니다.
+ 기관과 개인 간의 정보 격차: 실시간 데이터 분석 인프라를 갖춘 기관 투자자와 달리 개인 투자자는 악재 발생 시 즉각적인 대응이 불가능해 자산 손실 위험에 상시 노출되어 있습니다.
+ 단발성 정보의 학습화 부재: 뉴스는 단발성으로 소비되는 경향이 강해 특정ㄷ 산업의 흐름을 읽기로 이어지기 어렵습니다.
\이에 투자자가 효율적으로 정보를 소비할 수 있도록 돕는 시스템을 개발하게 됐습니다.
### 1.2. Goal
투자자가 각자의 수요에만 최적화된 뉴스만 수신할 수 있도록 해 정보 습득의 효율성을 개선합니다.

### 1.3. Target Market
복잡한 기사 전문을 읽기보다 AI가 선별한 핵심 포인트와 예상 impact 수치를 통해 학습하고자 하는 입문자\
실시간 뉴스 대응이 수익류롸 직결되는 개인투자자

## 4. Concept of operation

### 1) 관심사/키워드 지정

| Purpose | 사용자의 관심사에 맞는 데이터 등록 |
| :--- | :--- |
| Approach | 사용자가 디스코드 명령어를 통해 입력하면 DB에 저장 |
| Dynamics | 최초 서비스 이용 또는 변경 시 발생 |
| Goals | 개인별 맞춤형 뉴스 필터링의 기준점 |

### 2) AI Summary 

| Purpose | 뉴스의 실제 가치를 빠르게 판단할 수 있도록 도움 |
| :--- | :--- |
| Approach | 수집된 뉴스 본문을 LLM에 전달해 핵심 요약을 생성 |
| Dynamics | 매칭되는 키워드의 뉴스가 API에 감지될 때 자동 실행 |
| Goals | 정보 탐색 시간 단축 및 즉각적인 대응 지원 |

### 3) Contextual Learning

| Purpose | 단발성 정보가 아닌 거시적 흐름 학습 |
| :--- | :--- |
| Approach | 특정 종목이나 산업의 최근 뉴스 10개를 종합하여 AI가 전체적인 리포트를 작성 |
| Dynamics | 사용자가'/learning' 명령어를 입력할 경우 실행 |
| Goals | 투자자의 금융 문해력 향상 |

### 4) Risk Monitoring

| Purpose | 투자 위험도 경고 |
| :--- | :--- |
| Approach | 시스템이 실시간으로 수집되는 뉴스들의 impact score를 취합해 특정 종목에 부정적 기류가 임계치를 넘을 경우 경고 알림 생성 |
| Dynamics | 보유 종목 관련 악재 뉴스가 단기간에 집중 발생하거나 시장 변동성이 급증할 때 시스템에 의해 자동 실행 |
| Goals | 사용자가 인지하지 못한 리스크를 사전에 통보해 자산 손실 최소화 및 안전한 투자 환경 조성 |


## 5. Problem Statement
AI의 요약 결과가 원문과 다를 수 있는 '할루시네이션' 리스크가 존재해 이를 극복하기 위해 항상 원문 뉴스 기사 링크를 함께 제공합니다. 무료 API 사용 시 데이터 갱신 주기에 지연이 발생할 수 있습니다.

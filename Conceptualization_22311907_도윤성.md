## 1. Business purpose
### 1.1. Project background

'2024년 한국인의 수면 실태' 보고서에 따르면, 한국인의 평균 수면 시간은 6시간 58분으로 OECD 평균보다 18% 부족합니다. 수면 장애 및 불면증 진료 환자는 2010년 대비 약 140% 급증(약 67만 8,000명)하여 수면 건강이 개인의 문제를 넘어 국가적 차원의 관리 대상으로 부상했습니다. 보건복지부는 수면을 잘 취하기 위해선 일정한 생체리듬 유지를 강조하고 있습니다. 이는 단순히 수면량을 늘리는 것이 아닌 잠자리에 들고 일어나는 시간을 일정하게 유지하여 뇌 기능 회복과 심리적 안정에 결정적인 역할을 하기 때문입니다. 일반적으로 권장하는 적정 수면 시간이 있지만 유전적, 환경적 요인에 따라 적정 수면 시간은 모두 다를 수 밖에 없습니다. 타인이 정해준 평균에 맞추려는 시도는 오히려 사용자에게 압박감을 주어 수면의 질을 떨어뜨리는 결과를 초래하기도 합니다.

+ **나만의 Golden Time 추적**: 본 시스템은 타인이 정의한 평균 수치가 아닌, 사용자 본인이 직접 입력한 주관적 컨디션 데이터를 기반으로 분석을 수행합니다.
+ **데이터로 증명하는 규칙성**: 추상적인 권고를 넘어, 데이터 기반의 피드백을 제공함으로써 사용자가 스스로 건강한 생체리듬을 찾아가도록 돕는 것입니다.


### 1.2. Goal
사용자가 자신의 수면 패턴과 컨디션의 상관관계를 파악하여 최적의 수면 효율을 찾도록 돕는 시각화 시스템을 개발합니다. 


### 1.3. Target Market
수면 장애, 불면증을 가진 환자
본인의 적정 수면 시간을 알고 싶은 사용자

## 4. Concept of operation

### 1) 관심사/키워드 지정

| Purpose | 사용자의 관심사에 맞는 데이터 등록 |
| :--- | :--- |
| Approach | 사용자가 디스코드 명령어를 통해 입력하면 DB에 저장 |
| Dynamics | 최초 서비스 이용 또는 변경 시 발생 |
| Goals | 개인별 맞춤형 뉴스 필터링의 기준점 |

### 2) AI Summary 

| Purpose | 뉴스의 실제 가치를 빠르게 판단할 수 있도록 도움 |
| :--- | :--- |
| Approach | 수집된 뉴스 본문을 LLM에 전달해 핵심 요약을 생성 |
| Dynamics | 매칭되는 키워드의 뉴스가 API에 감지될 때 자동 실행 |
| Goals | 정보 탐색 시간 단축 및 즉각적인 대응 지원 |

### 3) Contextual Learning

| Purpose | 단발성 정보가 아닌 거시적 흐름 학습 |
| :--- | :--- |
| Approach | 특정 종목이나 산업의 최근 뉴스 10개를 종합하여 AI가 전체적인 리포트를 작성 |
| Dynamics | 사용자가'/learning' 명령어를 입력할 경우 실행 |
| Goals | 투자자의 금융 문해력 향상 |

### 4) Risk Monitoring

| Purpose | 투자 위험도 경고 |
| :--- | :--- |
| Approach | 시스템이 실시간으로 수집되는 뉴스들의 impact score를 취합해 특정 종목에 부정적 기류가 임계치를 넘을 경우 경고 알림 생성 |
| Dynamics | 보유 종목 관련 악재 뉴스가 단기간에 집중 발생하거나 시장 변동성이 급증할 때 시스템에 의해 자동 실행 |
| Goals | 사용자가 인지하지 못한 리스크를 사전에 통보해 자산 손실 최소화 및 안전한 투자 환경 조성 |


## 5. Problem Statement
AI의 요약 결과가 원문과 다를 수 있는 '할루시네이션' 리스크가 존재해 이를 극복하기 위해 항상 원문 뉴스 기사 링크를 함께 제공합니다. 무료 API 사용 시 데이터 갱신 주기에 지연이 발생할 수 있습니다.
